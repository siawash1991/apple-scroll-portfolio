# CLAUDE.md — Context اصلی پروژه

## معرفی پروژه
این یک وب‌سایت شخصی (Personal Portfolio) تک‌صفحه‌ای است که با الهام از سبک اسکرول‌محور اپل (مثل صفحات معرفی محصول در apple.com) ساخته می‌شود.

جلوه اصلی سایت (Hero Section):
- یک تصویر از چهره‌ی صاحب سایت به عنوان لایه‌ی بالایی نمایش داده می‌شود.
- با اسکرول کردن کاربر، این تصویر به دو نیمه (چپ و راست) تقسیم شده و به سمت بیرون حرکت می‌کند (Reveal Effect مشابه افکت‌های Product Reveal اپل).
- زیر این لایه، تصویر یک ربات (Robot) نمایان می‌شود که استعاره از "هوش مصنوعی / آینده تکنولوژی" است.
- افکت باید کاملاً scroll-linked باشد، نه صرفاً یک انیمیشن با تایمر.

## استک فنی
- HTML5 / CSS3 / Vanilla JavaScript (بدون فریم‌ورک سنگین)
- کتابخانه انیمیشن: GSAP + ScrollTrigger plugin
- بدون بک‌اند (سایت کاملاً استاتیک است)
- هدف دیپلوی نهایی: Vercel

## بخش‌های صفحه
1. **Hero Section** — افکت اصلی reveal چهره → ربات (اولویت اول پروژه)
2. **About Section** — معرفی کوتاه شخصی
3. **Projects Section** — نمایش نمونه‌کارها (grid یا کارت‌های ساده)
4. **Contact Section** — راه‌های ارتباطی (ایمیل، لینکدین، گیت‌هاب)

## قواعد و کانونشن‌های کدنویسی
- کد باید تمیز، خوانا و کامنت‌گذاری‌شده باشد.
- از نام‌گذاری معنادار برای کلاس‌های CSS استفاده شود (BEM ساده).
- هیچ کتابخانه‌ی سنگین یا فریم‌ورکی اضافه نشود مگر با تأیید صریح.
- کد باید responsive باشد (موبایل، تبلت، دسکتاپ).
- تصاویر باید بهینه بارگذاری شوند.
- هر تغییر مهم باید در یک commit جداگانه ثبت شود.

## نکات مربوط به افکت Scroll Reveal
- ویدیوی Hero توسط کاربر ساخته می‌شود؛ Agent فریم‌ها را با ffmpeg استخراج و در canvas پخش می‌کند.
- فایل `js/hero-scroll.js` شامل `FRAME_CONFIG` برای فعال‌سازی frame sequence است.
- از `gsap.timeline` + `ScrollTrigger` با `pin: true` و `scrub: true` استفاده شود.
- Placeholder CSS reveal تا آماده شدن ویدیو فعال است.
